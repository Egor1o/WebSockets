# WebSocket

Analyzing the performance of WebSockets

- Install dependencies:

```sh
   `npm install`
```

## To run and test socket server

- Start server:

```sh
  `npm run dev:sockets`
```
- Run tests:
```sh
  `npm run performance`
```

## or to run and test http server

- Start server:

```sh
  `npm run dev:http`
```

- Run tests:
```sh
  `npm run performance:http`
```
